如何在windows的Cmd中自动补全

发表于 2021-08-12 更新于 2021-08-19 分类于 技术

Windows命令行自动补全的功能,修改相当方便.

首先用命令里输入 regedit 打开注册表

然后将 HKEY_LOCAL_MACHINE\Software\Microsoft\ Command Prosessor\CompletionChar的键值改为9(十进制)

然后便可在命令行窗口用TAB键自动补全命令了

个性化配置

下载主题并改变网站样式

下载 Next 主题

进入主题配置文件 _config.yml 中找到 Scheme 字段，根据需要启动相应的scheme

配置页面中文

在 站点配置文件 _config.yml 中找到 language 字段，
并将其值改为 zh-CN

设置用户头像

在站点文件目录下 source 中创建 uploads 文件夹，并将头像图片放入其中，
之后在 主题配置文件 _config.yml 中找到 avatar 字段，
并将其url值改为 头像文件地址

设置暗色模式

在 主题配置文件 _config.yml 中找到 darkmode 字段，
并将其值改为 true

设置代码高亮主题

在 主题配置文件 _config.yml 中找到 highlight_theme 字段，
并将其值改为 night
