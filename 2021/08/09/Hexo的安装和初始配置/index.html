Hexo的安装和初始配置

发表于 2021-08-09 更新于 2021-08-19 分类于 技术

初次使用Hexo，需要经过几个步骤

必要配置

安装hexo
将 Hexo 所在的目录下的 node_modules 添加到环境变量之中即可直接使用 hexo ：
创建并初始化网站

个性化配置

下载主题并改变网站样式

下载 Next 主题

进入站点配置文件 _config.yml 中找到 theme 字段，将其更改为 next

进入主题配置文件 _config.yml 中找到 Scheme 字段，根据需要启动相应的scheme

配置页面中文

在 站点配置文件 _config.yml 中找到 language 字段，
并将其值改为 zh-CN

设置用户头像

在站点文件目录下 source 中创建 uploads 文件夹，并将头像图片放入其中，
之后在 主题配置文件 _config.yml 中找到 avatar 字段，
并将其url值改为 头像文件地址

设置暗色模式

在 主题配置文件 _config.yml 中找到 darkmode 字段，
并将其值改为 true

设置代码高亮主题

在 主题配置文件 _config.yml 中找到 highlight_theme 字段，
并将其值改为 night
